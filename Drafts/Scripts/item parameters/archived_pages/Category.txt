{{About|[[craftRecipe]] <code>category</code> parameter|PZwiki categories|:Category:Main}}
The <code>category</code> parameter is used to define the category of the recipe, thus helping users find the craft. The category is defined by the following format:
{{CodeSnip
| lang = lua
| code =
category = <category name>,
}}
