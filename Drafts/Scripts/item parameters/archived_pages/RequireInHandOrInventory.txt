==== RequireInHandOrInventory ====
The parameter specifies the item that the player must have in order to use the original item. You can select multiple items.
<syntaxhighlight lang="Java">
RequireInHandOrInventory = Matches/Matchbox/LighterDisposable/Lighter,
</syntaxhighlight>
