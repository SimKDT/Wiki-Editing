The {{Code|muscleStrainSkill}} parameter is used to define the skill that reduces the muscle strain generated during a timed action. It is defined by a skill name. To use a muscle strain skill in a timed action, use the following format:
{{CodeSnip
| lang = lua
| code =
muscleStrainSkill = <skill name>,
}}

== Example ==
{{CodeSnip
| lang = lua
| code =
muscleStrainSkill = Strength,
}}
