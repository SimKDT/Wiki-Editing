==== MultipleHitConditionAffected ====
Parameter for weapons. If the parameter is set to true, then when a weapon hits targets, the durability of the weapon is spent. Usually set to false for firearms.
<syntaxhighlight lang="Java">
MultipleHitConditionAffected = false,
</syntaxhighlight>
