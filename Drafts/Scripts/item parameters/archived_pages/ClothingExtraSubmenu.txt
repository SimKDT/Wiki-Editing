==== ClothingExtraSubmenu ====
Used for clothes. The parameter specifies an additional option for the context menu of how the item can be equipped by default.
<syntaxhighlight lang="Java">
ClothingExtraSubmenu = ForwardCap,
</syntaxhighlight>
